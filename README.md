# Simple Power BI Dashboard

## Overview

This repository contains my first Power BI dashboard project. The dashboard was created to practice data visualization, report design, and business intelligence concepts using Microsoft Power BI.

## Features

* Interactive dashboard visualizations
* Data analysis and insights
* User-friendly report layout
* Basic KPI and performance tracking

## Files

* `HR dashboard.pbix` – Power BI dashboard file

## Tools Used

* Microsoft Power BI

## Purpose

The purpose of this project is to improve my Power BI skills and gain hands-on experience in creating dashboards for data analysis and decision-making.
